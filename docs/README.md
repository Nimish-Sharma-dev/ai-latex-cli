# docs/ — Project Documentation

This directory contains the technical and user-facing documentation for the AI LaTeX CLI project.

---
## Reading Order

**New to the project?** Read in this order:

1. `../README.md` — Project overview, quick start, and installation
2. `COMMAND_REFERENCE.md` — Learn what commands are available and how to use them
3. `ARCHITECTURE.md` — Understand how the system works internally

**Contributing code?** Read:

1. `ARCHITECTURE.md` — Understand the layer boundaries before writing code
2. `../CONTRIBUTING.md` — Branching strategy, PR rules, and coding standards
3. `../src/README.md` and each sub-package's `README.md`

---

## Documentation Standards

When adding new documentation to this directory:

- Use clear Markdown with a consistent heading hierarchy (`#`, `##`, `###`)
- Include code examples for every configuration option and command
- Use tables for structured reference information (command options, config variables, etc.)
- Update this `README.md` whenever a new document is added
